# CTRXL DRACIN - Redesign Notes

## Current Design Analysis

### Strengths:
- Dark theme is good for streaming platform
- Ken Burns animation on hero is nice
- Card hover effects are smooth
- Bottom navigation is functional

### Issues to Fix:
1. **Hero section** - Too dark, lacks visual appeal
2. **Source tabs** - Look outdated with emoji icons
3. **Drama cards** - Too small, need better visual hierarchy
4. **Color scheme** - Pure black is too harsh, needs warmth
5. **Typography** - Bebas Neue for display is too aggressive
6. **Bottom nav** - Looks basic, needs glass morphism upgrade
7. **Search page** - Empty state needs improvement
8. **Detail page** - Episode grid looks cluttered
9. **Bookmarks page** - Empty state is boring

## Redesign Plan

### New Design Direction: "Premium Streaming Experience"
- Inspired by Netflix, Disney+, Apple TV+
- Gradient backgrounds with subtle depth
- Glassmorphism for overlays
- Smooth micro-interactions
- Better card designs with ratings
- Modern typography with Inter/Geist
